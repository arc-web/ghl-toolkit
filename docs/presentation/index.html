Get started

Up and running fast

1
Get your Private Integrations key - In GHL: Settings → Integrations → Private Integrations. Select scopes for contacts, conversations, opportunities, calendars, and whatever else you need.
2
Pick your runtime - Clawdbot for the easiest AI assistant path, mcporter for a lightweight CLI, or deploy the server directly to Vercel/Railway/Render.
3
Wire it to Claude Desktop - Add the MCP server to your Claude config, paste in your Private API Key and Location ID, and reload.
4
Start automating - Try a prompt like "find cold leads from this week and draft follow-up SMS" and watch the CRM do the work.
